## Features

- About
- Blog
- Post Acrhive
- Category
- Tag Cloud
- Resources Viewer

## Installation

- Fork this repository
- Clone your forked repository

```bash
git clone https://github.com/you/Materialize-Jekyll.git 
```

## Usage

- Modify `_config.yml` file.
- Create your post on `_posts` directory.
